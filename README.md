# MZS · Portfolio

Personal portfolio website of **Muhammad Zeeshan Siddiqui** — CS Student at SSUET, Full-Stack Developer, and AI/ML enthusiast.

🔗 **Live site:** [zeeshan-siddiqui-portfolio.vercel.app](https://zeeshan-siddiqui-portfolio.vercel.app/)

---

## ✨ Features

- Animated hero section with typewriter effect
- Custom canvas cursor & ambient particle background (desktop)
- Smooth scroll-reveal animations
- Filterable project showcase with detail modals
- Animated education timeline
- Animated skill proficiency bars
- Fully functional contact form (powered by Formspree)
- Fully responsive — optimized for mobile, tablet, and desktop

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, glassmorphism, responsive design)
- Vanilla JavaScript (Canvas API, Intersection Observer, Fetch API)
- [Formspree](https://formspree.io/) for contact form submissions

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/ZEESHANSIDDIQUIgit/your-repo-name.git
   cd your-repo-name
   ```

2. Open `index.html` directly in your browser, or serve it locally:
   ```bash
   npx serve .
   ```

3. To enable the contact form, replace the Formspree endpoint in `index.html`:
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST" ...>
   ```
   with your own form ID from [formspree.io](https://formspree.io/).
